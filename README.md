🌙 SleepTracker Pro — трекер сна с анализом циклов
Интеллектуальный трекер сна для анализа циклов, фаз и качества отдыха.
Поддерживает расчёт фаз сна (быстрый/медленный), рекомендации по улучшению, статистику за период и сохранение данных.
Реализован на 7 языках программирования для демонстрации подходов к анализу временных рядов и работы с данными.

🧠 Концепция
SleepTracker Pro — это инструмент для анализа качества сна на основе времени засыпания и пробуждения. Он позволяет:

✅ Анализировать циклы сна (каждый цикл ≈ 90 минут) с выделением фаз быстрого и медленного сна.

✅ Оценивать качество сна по шкале от 1 до 10 на основе длительности, глубины и регулярности.

✅ Сохранять историю записей с датами и показателями.

✅ Строить статистику за последние 7 дней: среднее время сна, качество, количество циклов.

✅ Давать рекомендации по улучшению сна (регулярность, проветривание, отсутствие гаджетов).

✅ Экспортировать данные в CSV и JSON.

✅ Работать как в интерактивном режиме, так и через аргументы командной строки.

🚀 Как запустить
Каждая версия самодостаточна. Установите зависимости (если есть) и запустите.

Python
bash
pip install matplotlib
python sleep_tracker_python.py
C++
bash
g++ -std=c++17 sleep_tracker_cpp.cpp -o sleeptracker
./sleeptracker
Java
bash
javac sleep_tracker_java.java && java sleep_tracker_java
C# (.NET Core)
bash
dotnet run
Go
bash
go run sleep_tracker_go.go
Rust
bash
cargo build --release && ./target/release/sleep_tracker_rs
JavaScript (Node.js)
bash
node sleep_tracker_js.js
